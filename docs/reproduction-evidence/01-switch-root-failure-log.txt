Root Cause
----------------------------------------------------------------------
The initrd was rebuilt by dracut --rebuild but WITHOUT --add bootc.
The bootc dracut module (51bootc) has check() { return 255; }, meaning
dracut NEVER includes it automatically. Without the module:
  - bootc-root-setup.service is absent from the initramfs
  - The composefs EROFS overlay is never assembled
  - /sysroot remains the raw XFS partition (no /etc/os-release)
  - systemctl switch-root /sysroot fails with "os-release file is missing"

This was confirmed via lsinitrd on the ESP initrd — zero bootc-related files.
